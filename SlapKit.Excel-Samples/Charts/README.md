# SlapKit.Excel Chart Samples

A collection of sample projects demonstrating [SlapKit.Excel](https://slapkit.com/excel/)'s charting capabilities. Those can be used in conjunction with [the documentation](https://docs.slapkit.com/excel/documentation/features/charts/charts-overview.html) and as a starting point for charting solutions.
